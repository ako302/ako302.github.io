I come across Hexo when I want to record my journal of learning code. I’m going to show you how I start this blog with hexo in Windows.

What is Hexo then?

Hexo is a powerful blog framework. Upon installing, we can create posts immediately without writing HTML, JS, etc. However, if you need custom themes, you will still have to write your own code~

Hexo’s posts are written in Markdown and it is very easy to write. Here is one of the markdown cheatsheets: markdown here
